Sudoku Solver


"Solve any sudoku puzzle through brute force and nothing else."


Download

To use this, make sure you have python, cv2, and flask installed on your computer.

Run with python3 sudoku.py in your command line and go to the link provided by the server.


Info

This Sudoku Solver can solve 4x4, 6x6, and 9x9 puzzles.

Some 9x9 irregular puzzles might fail.

I made this for my Artificial Intelligence class when we were learning Constraint Satisfaction Problems.

This app also introduced me to Flask, which is what I like to use for GUIs.


The Algorithm

AC-3: Arc Consistency Algorithm #3 is the most common algorithm taught for solving constraint problems.

Wikipedia Page for AC-3


The Heuristic

Minimum Remaining Values: The least amount of values in a cell was selected during each parse of the backtracking.
