Automata Tester


"Finite Machine gaming"


Download

To use this, make sure you have python installed on your computer.

Run with python3 main.py in your command line and go to the link provided by the server.


Info

Project is currently unfinished.

Takes in Automata Machines as JSONs.

You can do a variety of operations to the machine.


Operations

Test strings up to a certain length.

Convert NFA to DFA.

Combine two DFAs with the same language (returns DFA).

Combine any two FAs (returns NFA)

Remove unused states from a machine.


Not implemented but planned

Convert FA to regex.

Convert PDA to CFG.
