Acid Trip


My image deepfrier/cooker/fuck-uper.

This is a fraction of what it can do.


Download

To use this, make sure you have python, cv2, and flask installed on your computer.

Run with python3 acidtrip.py in your command line and go to the link provided by the server.

To use a new image, place it into the input folder and select it in the application.


Usage Tips

A higher amount takes longer, as well as the size of the image.

If you are impatient, reduce the size of the image.

The static output folder has a cache of sorts.

If you want to build up a cache of a single image, using the multiple option will help with that.


Backstory

One day, I decided to work on my computer vision project that was due.

The first part of the project involved writing a convolution function instead of the one provided by cv2.

On my first test, I had forgotten to multiply the value in the matrix to the pixel, which blew up the image.

I liked the result so much I made an application out of it.
